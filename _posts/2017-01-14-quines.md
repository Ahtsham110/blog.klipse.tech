## Scheme

## Python

Python's `%r` format conversion uses the `repr()` function to return a string containing the source code representation of its argument:


~~~klipse-python
x = 'x = %r\nprint(x %% x)'
print(x % x)
~~~

With the new `str.format`:


~~~klipse-python
x = 'x = {!r};print(x.format(x))';print(x.format(x))
~~~
